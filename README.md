web-storage使用文档
=======
web-storage是一款web端的缓存工具,基于html5的storage进行扩展.
1. 添加了过期时间,传统的storage不支持设置过期时间,需要用户自行清除
2. 数据加密,可以对key,value进行加密,加密是可逆的,加密只是最基本的显示安全
3. 设置key前缀,可以为key字段设置一个前缀
4. 支持存放string以外的数据类型.传统的storage只能存放string
